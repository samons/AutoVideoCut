# AutoVideoCut
Automatically video highlights based on audio and video cues. 

1. prerequisite pkgs 
    1. fftw-3.3.5-dll64
    2. boost_1_63_0
    3.opencv3.2
    4. Qt5.9.1
    5. ffmpeg-20170503-a75ef15-win64-dev

2. Build in VS2017
   a. Edit ./src/vs2017/AutoCut.props based on your environment
   b. open ./src/vs2017/AutoCut.sln by vs2017
