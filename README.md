# AutoVideoCut
Automatically video highlights based on audio and video cues. 

1. Prerequisite pkgs 
    - fftw-3.3.5-dll64
    - boost_1_63_0
    - opencv3.2
    - Qt5.9.1
    - ffmpeg-20170503-a75ef15-win64-dev

2. Build in VS2017
   - Edit ./src/vs2017/AutoCut.props based on your environment
   - Open ./src/vs2017/AutoCut.sln by vs2017
